Cardify App

Cardify is a Flask-based web application designed to assist you in creating, managing, and utilizing flashcards for learning, memorization, planning tasks, etc.

Features

  • Create flashcards with custom fields: title, content, hint, notes, and image.
  • Flashcards can be parents of other flashcards.
  • Generate cards using the open-ai model "gpt-4o".
  • Generate images for the cards using the open-ai model "dall-e-3".
  • Create topics (topics can be parents of other topics).
  • Create decks for each topic and add flashcards to them.
  • Visualize topics, decks, and flashcards in a graphical way using the python package "pyvis".
